# Schaufler_Azure_Anleitung

## Zusammenfassung
Kurzanleitung für die Schaufler Tooling GmbH & Co. KG zur Nutzung des Azure Portals: Beschreibt, wie Token-Verbrauch des KI-Agenten im Microsoft Foundry Portal eingesehen wird und wie Azure OpenAI Rechnungen im Azure Portal abgerufen werden können.

## Vollständiger Inhalt
# Kurzanleitung Azure Portal

Schaufler Tooling GmbH & Co. KG

## Token-Nutzung einsehen

Hier können Sie den Token-Verbrauch des KI-Agenten betrachten. Öffnen Sie dazu das Microsoft Foundry Portal unter ai.azure.com und gehen Sie wie folgt vor:

1 Klicken Sie in der linken Leiste auf **Bereitstellungen**.
2 Wählen Sie die Bereitstellung **gpt-5.4-mini** aus.
3 Klicken Sie oben auf den Tab **Metriken**. Dort sehen Sie Anfragen, Prompt-Token und Abschluss-Token für einen frei wählbaren Zeitraum.

## Rechnungen einsehen

Hier können Sie auf die Abrechnungen von Azure OpenAI zugreifen. Öffnen Sie dazu das Azure Portal unter portal.azure.com und gehen Sie wie folgt vor:

1 Klicken Sie auf **Abonnements**.
2 Wählen Sie **Azure subscription 1** aus.
3 Klicken Sie in der linken Leiste auf **Abrechnung**.
4 Wählen Sie dort **Rechnungen des Abrechnungsprofils**. Alle Rechnungen sind chronologisch aufgelistet und als PDF herunterladbar. Hinweis: Kosten erscheinen mit bis zu 48 Stunden Verzögerung.
