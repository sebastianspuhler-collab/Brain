# Server Hosting KI-Agend (1)

Transkript eines Meetings vom 17.04. zwischen Sebastian Spuhler, Jonas Rösch und Amin Douioui bezüglich der Server-Hosting-Infrastruktur für den KI-Agenten beim Kunden Schaufler. Besprochen werden die aktuellen Server-Ressourcen (16GB RAM, 200GB Speicher) sowie eine langfristige Migration auf eine Cloud-Lösung. Zudem werden die Anforderungen durch Proleis-Abfragen des Beschaffungsagenten analysiert.
