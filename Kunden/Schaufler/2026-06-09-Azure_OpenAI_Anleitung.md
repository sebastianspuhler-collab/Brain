# Azure_OpenAI_Anleitung

Interne Einrichtungsanleitung von Prozessia für die Konfiguration von Azure OpenAI und der Microsoft Graph API in Unternehmen. Das Dokument beschreibt in zwei Teilen die Schritte zur Einrichtung einer Azure OpenAI Ressource inklusive Sprachmodell sowie die Installation der Graph API für den Zugriff auf Microsoft 365 Unternehmensdaten. Es handelt sich um eine kundenspezifische technische Dokumentation, die dem Kunden Schaufler zugeordnet ist.
