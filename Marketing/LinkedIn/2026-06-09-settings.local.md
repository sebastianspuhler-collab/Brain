# settings.local

Lokale Claude-Berechtigungskonfiguration (settings.local.json) für ein Branding-Projekt. Die Datei definiert erlaubte Tool-Berechtigungen für den Claude AI-Agenten, darunter Web-Suche, Node.js-Ausführung und Dateizugriffe. Ein Bash-Befehl referenziert explizit ein LinkedIn-Posting-Skript mit dem Titel '5 Claude Skills I Can't Live Without', was auf Content-Erstellung für LinkedIn hindeutet.

## Vollständiger Inhalt
[JSON-Datei] {
  "permissions": {
    "allow": [
      "WebSearch",
      "Bash(npm install *)",
      "Bash(node ideas.js)",
      "Read(//usr/local/**)",
      "Read(//opt/homebrew/**)",
      "Bash(command -v node)",
      "Bash(find /Users/sesp01-user -name \"node\" -type f 2>/dev/null | head -5; find /usr -name \"node\" -type f 2>/dev/null | head -5; ls /Users/sesp01-user/.nvm 2>/dev/null)",
      "Read(//Users/sesp01-user/**)",
      "Read(//usr/**)",
      "Bash(sed -i '' \"43s/.*/const DOC_TITLE = \\\\\"5 Claude Skills I Can't Live Without\\\\\";/\" schedule-linkedin-post.js)",
      "Bash(sed -n '43p' schedule-linkedin-post.js)"
    ]
  }
}
