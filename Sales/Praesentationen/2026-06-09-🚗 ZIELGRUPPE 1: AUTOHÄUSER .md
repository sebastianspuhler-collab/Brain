# 🚗 ZIELGRUPPE 1: AUTOHÄUSER 

Dieses Dokument beschreibt ein Voice Agent Konzept speziell für Autohäuser als Zielgruppe. Es enthält eine Wirtschaftlichkeitsanalyse (ab wann sich ein Voice Agent lohnt), konkrete Einsatzszenarien (Terminvereinbarung, Probefahrtbuchung, Rückrufmanagement, After-Sales) sowie drei Leistungsmodelle mit Preisstruktur. Das Dokument dient als Sales-Grundlage für die Ansprache von Autohäusern als potenzielle Neukunden.

## Vollständiger Inhalt
🚗 ZIELGRUPPE 1: AUTOHÄUSER — Voice Agent Konzept     1️⃣ Untergrenze – Ab wann lohnt sich ein Voice Agent im Autohaus? Autohäuser haben starke Telefonlasten zu Stoßzeiten (Morgens, Montage, Aktionstage).
 Für eine klare Entscheidung: ⭐ Der Agent lohnt sich ab ca. 10–15 Anrufen pro Tag. Das gilt für: Werkstatt-Service
 Probefahrtanfragen
 Rückrufwünsche
 Verfügbarkeitsanfragen
 Unter 10 Calls/Tag
 ➡️ Lohnt sich nicht, menschliche Bearbeitung ist günstiger. Über 10–15 Calls/Tag
 ➡️ Deutliche Entlastung + weniger verpasste Aufträge. 20+ Calls/Tag
 ➡️ Sehr guter Fit, typische Autohausgröße.  2️⃣ Kernprobleme von Autohäusern (Dein Sales-Hebel) Verpasste Anrufe = verlorene Serviceumsätze
 Werkstattteam oft überlastet → lange Wartezeiten
 Unqualifizierte Anrufe halten Mitarbeiter auf
 Kunden rufen außerhalb der Öffnungszeiten an
 Probefahrten/Service-Termine kosten viel Zeit für Abwicklung
 Papierlisten/Kalender-Chaos ohne Standardisierung
 ➡️ Der Agent löst alle 6 Punkte.  3️⃣ Konkrete Einsatzszenarien (realistische Branchenfälle) 📞 Szenario 1: Service-Terminvereinbarung Problem:
 Montagmorgen: 30 Anrufe, kaum jemand erreichbar. Voice Agent kann: Kennzeichen & Modell abfragen
 Serviceart klären (TÜV, Inspektion, Ölwechsel…)
 Termine vorschlagen
 Direkt einbuchen (Kalender/CRM)
 SMS-Bestätigung senden
 Nutzen:
 ⟶ Werkstatt ist permanent ausgelastet
 ⟶ Kein Anruf geht verloren  🚗 Szenario 2: Probefahrt-Buchung Problem:
 Probefahrten sind umständlich zu koordinieren. Agent fragt ab: Modellwunsch
 Führerschein vorhanden?
 Wunschdatum
 Datenaufnahme
 Terminbuchen
 Nutzen:
 ⟶ Verkauf bekommt qualifizierte Leads
 ⟶ Weniger telefonischer Aufwand  🧾 Szenario 3: Verfügbarkeitsanfragen Kunden fragen: „Ist der Golf VIII verfügbar?“
 „Ist das Auto noch da?“
 „Wann kommt mein Fahrzeug an?“
 Agent kann:
 → Bestand abfragen
 → Lieferstatus aus CRM abfragen (optional)
 → Infos + Follow-up senden  📲 Szenario 4: Rückruf-Management „Bitte rufen Sie mich zurück“
 → Agent sammelt Daten
 → Sendet strukturierten Rückruf an Service oder Verkauf Nutzen:
 ⟶ Keine Notizzettel mehr
 ⟶ Kein vergessener Rückruf  ⛑ Szenario 5: After-Sales & Inspektionserinnerungen Outbound (optional): Kunden ankommender Inspektion erinnern
 TÜV fällig → Termin anbieten
 Angebotsaktionen durchrufen
 Nutzen:
 ⟶ Mehr Aufträge im After-Sales
 ⟶ Automatisierung von Umsatzbringern  4️⃣ Einheitliche Leistungsmodelle für Autohäuser (3 Stufen) Basierend auf deinem bestehenden Preismodell.  🔹 Modell 1 – Basic Agent (Sekretariat) Preis: 800 € Setup + Minutenpreis
 Einsatz: Für Autohäuser mit bis zu ca. 20–40 Calls/Tag Leistet: Begrüßung
 Anliegenaufnahme
 Weiterleitung (Service / Verkauf)
 Rückrufannahme
 Öffnungszeiten-/FAQ-Infos
 ➡️ Ideal für den Einstieg.  🔹 Modell 2 – Workflow Agent (Service & Probefahrt) Preis: 2.000 € Setup + Minutenpreis + Wartung
 Einsatz: Autohäuser mit wiederkehrenden Prozessen Leistet: Service-Terminabfrage (Modell, Kennzeichen, Datum)
 Probefahrtbuchung
 Annahme von Reparaturwünsche
