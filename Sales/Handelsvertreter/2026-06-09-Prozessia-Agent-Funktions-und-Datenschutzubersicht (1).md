# Prozessia-Agent-Funktions-und-Datenschutzubersicht (1)

Das Dokument beschreibt den Prozessia Agent, ein KI-gestütztes System zur Unterstützung von Mitarbeitenden im Arbeitsalltag. Es erläutert die zentralen Funktionen wie Dokumentenverarbeitung, Texterstellung und Informationsrecherche sowie die Möglichkeit zur individuellen Personalisierung der Assistenten. Zusätzlich werden Datenschutz- und Datensicherheitsaspekte behandelt, darunter EU-exklusive Datenverarbeitung und Zero-Knowledge-Architektur.

 Was ist der Prozessia Agent?
Kernfunktion
Der Prozessia Agent ist ein KI -gestütztes System, das auf einem 
Server oder VPS des Kunden betrieben wird. Es nutzt interne 
Unternehmensdokumente, um Mitarbeitende bei verschiedenen 
Aufgaben zu unterstützen.
Einsatzgebiet
Das System vereinfacht Arbeitsprozesse durch schnellen Zugriff 
auf Informationen und Unterstützung bei Recherche, Texterstellung 
und Dokumentenaufbereitung.
 Zentrale Funktionen
Antworten auf Fragen
Liefert präzise Antworten auf Basis 
interner Unternehmensdokumente
Texterstellung
Erstellt und überarbeitet Texte, E -
Mails, Schreiben und 
Zusammenfassungen
Informationsrecherche
Unterstützt bei der Suche und 
Strukturierung von Inhalten
Der Agent nutzt automatisch vorhandenes Unternehmenswissen und kann für verschiedene Rollen und Abteilungen angepasst 
werden. Dies ermöglicht eine effiziente Informationsverarbeitung ohne manuelle Suche in Dokumenten.
 Arbeiten mit dem Prozessia Agent
01
Anfrage stellen
Mitarbeitende stellen Fragen wie in einem Chat
02
Antwort erhalten
Der Agent liefert Antworten basierend auf hinterlegten Dokumenten
03
Dokumente erstellen
Erstellung von Dokumententwürfen wie interne Schreiben oder Vorlagen
04
Inhalte aufbereiten
Zusammenfassungen längerer Texte und Hervorhebung relevanter Informationen
Das System eignet sich besonders für wissensintensive Aufgaben und standardisierte Abläufe 
im Unternehmensalltag. Dokumentenbezogene Funktionen
Umfassende Dokumentenverarbeitung
•Zugriff auf alle im System hinterlegten Dokumente
•Extraktion relevanter Inhalte für präzise Antworten
•Generierung von Zusammenfassungen komplexer Unterlagen
•Formulierung von Texten im gewünschten Stil
•Erstellung neuer Dokumente auf Basis vorhandener Informationen
Der Prozessia Agent erleichtert den Umgang mit umfangreichen Akten, 
Richtlinien oder Unterlagen erheblich.
 Personalisierung der Assistenten
Individuelle Assistenten
Jeder Mitarbeitende erhält 
einen eigenen KI -Assistenten, 
der an spezifische Rollen und 
Verantwortlichkeiten angepasst 
werden kann.
Flexible Konfiguration
Unterschiedliche 
Aufgabengebiete und 
Schreibstile sind möglich. 
Zugriff auf für die jeweilige 
Person relevante Dokumente 
wird individuell eingerichtet.
Konsistente Qualität
Einheitliche Qualität bei der 
Dokumentenerstellung über alle 
Abteilungen und Mitarbeitende 
hinweg.
 Datenschutz und Datensicherheit
EU-exklusive 
Datenverarbeitung
Alle Daten werden ausschließlich in 
der EU verarbeitet. Nutzung der 
Microsoft EU Data Boundary 
garantiert, dass keine Übertragung 
von Daten außerhalb Europas erfolgt.Zero Knowledge Architektur
Prozessia hat keinen Zugriff auf 
Kundendaten. Dokumente verbleiben 
vollständig beim Kunden. Daten 
werden nicht für das Training von KI -
Modellen verwendet.Datensensible Bereiche
